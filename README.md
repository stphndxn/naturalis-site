# Naturalis landing page

Static landing page prepared for Netlify deployment.

## Files

- `index.html`: page structure and Netlify form markup
- `styles.css`: editorial visual system, responsive layout, and warm dark mode
- `netlify.toml`: publish directory for Netlify

## Deploy on Netlify

1. Push this directory to a GitHub repository.
2. In Netlify, choose **Add new site** -> **Import an existing project**.
3. Connect the GitHub repo and keep the default publish directory as the repo root.
4. After the first deploy, open the site dashboard and confirm the `naturalis-waitlist` form appears under **Forms**.
